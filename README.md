#Gentrificación - Indicadores Urbanos

Dashboard interactivo en Power BI que analiza la gentrificación y algunos de sus indicadores urbanos en México y Colombia, con enfoque en Comala, Colima. Integra variables de vivienda, movilidad, desigualdad, competitividad e inversión, a través de mapas, gráficas dinámicas y segmentación temporal interactiva.
